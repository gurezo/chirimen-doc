## adb のセットアップ
### android-tools-adbをインストールします。
- Ubuntu 12.10以降
  ```
  $ sudo apt-get install -y android-tools-adb
  ```
- Fedora 19以降
  ```
  $ sudo yum install android-tools -y
  ```
- Mac OSX以降（[Homebrew](https://brew.sh/index_ja.html) を使っている時）
  ```
  $ brew install android-platform-tools
  ```
